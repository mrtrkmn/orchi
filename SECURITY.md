## Reporting security issues

If you notice/discover a security issue, please bring it to our attention right away!

Please DO NOT file a public issue, instead send your report privately to the repository maintainers via [GitHub Security Advisories](https://github.com/mrtrkmn/orchi/security/advisories/new).

Security reports are greatly appreciated and we will publicly thank you for it, although we keep your name confidential if you request it.
